PharmaSys Wholesale Hub
Project Overview
PharmaSys is a digital wholesale pharmacy management hub developed to modernize supply chain operations. This project focuses on improving efficiency in pharmaceutical procurement through automation and data-driven management.

Key Functionalities
Inventory Synchronization: Real-time tracking of medical stock levels to ensure consistent availability.

Batch Tracking: Precise monitoring of product batches to ensure quality control and regulatory compliance.

Order Fulfillment: An automated system designed to streamline the order-to-delivery process for wholesale clients.

Technical Stack
Core Development: ASP.NET and Visual Basic.

Database Management: SQL Server for secure and reliable data handling.

Project Contribution
As the developer of this system, I was responsible for the end-to-end architecture and implementation, focusing on:

Building robust backend logic to manage wholesale inventory transactions.

Ensuring the system provides accurate, real-time data synchronization.
